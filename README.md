# Smart Contract Summary & Q&A Assistant

**Educational RAG prototype** — upload PDF/DOCX contracts (NDAs, service agreements, etc.), get a quick summary, and ask natural-language questions with **source citations**.

**Important Disclaimer**  
This is a **demonstration / learning project**.  
**It is NOT legal software and MUST NOT be used as legal advice.**  
Treat all answers as potentially incomplete or incorrect.

## Quick Start — Run the Application in < 10 minutes

### 1. Prerequisites

- Python **3.10** – **3.12** (3.11 recommended)
- Git (optional — you can also download ZIP)
- Internet connection (needed for `pip install` and OpenRouter API)

### 2. Clone or download the project

```bash
git clone <[https://github.com/z3yad30/Contract-Summary-Q-A-Assistant]> smart-contract-rag-assistant
cd smart-contract-rag-assistant
